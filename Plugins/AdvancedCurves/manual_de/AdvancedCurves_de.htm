Advanced Curves v0.3

Funktionen / Features

  • Einführung eines neuen Objektes /Objekttyp: AdvCurves.

  • Beibehaltung der (Rückwärts-) Kompatibilität zu bestehenden Kurven - alle bestehenden Kurven Werkzeug, wie Array, Tube, Lathe, Extrude können weiter genutzt werden.

  • AdvCurves werden mit einer nicht geglätteten Außenlinie outline dargestellt: Also gerade Linien von Punkt zu Punkt.

  • Geglättete Kurven werden öfter unterteilt um ein wirklich glattes Aussehen zu bekommen.

  • AdvCurves werden jetzt vollständig in einem eigenen Editor erzeugt mit Hilfe des neuen Extend Curve Tool.

  • Genaues Fangen der Gitterpunkte.

  • Editor Fenster: Ein situationsabhängiges Menü (context menu) enthält alle Funktionen des normalen Kurven Menüs.

  • Ein Pfeil an einem Ende der Kurve zeigt deren Richtung an - das ist wichtig beim Skinning (überziehen) und beim Extrudieren um den Anfang der Kurve zu bestimmen.

Arbeiten mit AdvCurves

Erstellung

Nach der Auswahl des neuen Toolicons (), können Sie irgendwo in die Szene klicken und damit eine neue AdvCurve an diesem Punkt erstellen. Das ist einer der Unterschiede zu normalen Kurven, die komplett im Hauptfenster erstellt werden. Anstelle dessen wird eine "Nulllängen" Kurve erstellt. Sie können dieses Objekt dann editieren mit einem Doppelklick auf das Objekt in der Objektliste, oder Rechtsklick -> Objekt bearbeiten. Es wird das Editor Fenster der Kurve dadurch geöffnet.

Wie man eine Advanced Curve zeichnet

Nachdem Sie nun das Editor Fenster vor sich haben, sehen Sie die Kurve - oder ein leeres Fenster so es sich um eine neue und damit leere (Nulllängen) Kurve handelt. Außerdem sehen Sie ein neues Werkzeug unten in der Werkzeugleiste:

Dieses neue Werkzeug erlaubt es neue Punkte (Vertices im AoI Jargon) jederzeit hinzuzufügen. Man braucht nun die Kurve nicht mehr zu unterteilen, um neue Punkte zu erschaffen. Mit AdvCurce wählen Sie nur das Ende oder den Anfang der Kurve und klicken irgendwohin (sinnvollerweise dorthin wo Sie einen neuen Kurvenpunkt erschaffen wollen): An dieser Stelle wird ein neuer Punkt erschaffen, der sofort mit der Urspungskurve verbunden ist - also den Kurvenzug ergänzt / verlängert.

Da das bekannte Unterteilen der Kurvenpunkte ("Subdivide Selection") einen Einfluss auf alle anderen Kurvenpunkte hat, erlaubt das Extend Curve Tool das auch Einfügen neuer Kurvenpunkte an jeder beliebigen Stelle..Selektieren Sie einfach die beiden Punkte zwischen denen Sie einen Punkt erstellen wollen und klicken Sie auf eine beliebige Stelle (sinnvollerweise irgendwo zwischen den Punkten). Das fügt einen Kurvenpunkt hinzu, ohne die anderen Punkte zu beeinflussen. Bitte beachten Sie das zwischen dem letzten und ersten Punkt einer Kurve kein Punkt gesetzt werden kann - auch wenn die Kurve geschlossen ist! (Das geht nur wenn die Kurve nur 2 Punkte insgesamt hat). Das Werkzeug weiß dann einfach nicht wo der Punkt addiert werden muss - vor dem ersten oder hinter dem letzten  Punkt (was de facto dasselbe wäre). Wenn Sie ihrer - womöglich geschlossenen - Kurve zwischen Start und Endpunkt weitere hinzufügen wollen, tun Sie einfach so als wäre die Kurve offen und selektieren nur einen Punkt und fügen einen weiteren hinzu. 

Entfernungen werden in selbiger Art und Weise wie bei der Erstellung von Bones gehandhabt. Wenn Sie einen Punkt (Vertex) in der Top Ansicht erstellen, wird sein Y-Wert (in dem Fall die Höhe) denselben Wert haben wie der vorhergehende Punkt.

Der erste und letzte Punkt einer Kurve sind jeweils "gehighlighted" um eine bessere Übersicht zu erhalten.

Anmerkungen zu Nulllängen Kurven

Wenn eine neue AdvCurve zur Szene hinzugefügt wird ist sie zunächst mal leer. Das bedeutet das dort einfach keine Punkte in der Kurve vorhanden sind.

Seien Sie deswegen darauf vorbereitet das nicht alle anderen Tools und Plugins die Länge der Kurve analysieren - sie gehen vielmehr davon aus, das eine Kurve aus mindestens 2 Punkten besteht (denn ansonsten ist es eigentlich keine Kurve). Das kann dann zu einem "Alarm" führen wenn Sie ein Tools ausführen an einer Kurve mit weniger als 2 Punkten.

Deshalb - wenn Sie ein Programmierer sind und ein neues Werkzeug planen das mit Kurven umgeht, sollten Sie immer die Länge einer Kurven prüfen bevor Sie mit deren Punkten arbeiten.

Benutzen des Gitters (Raster/Grid)

Neue Punkte, die mit dem Extend Curve Tool erzeugt werden, werden automatisch - den aktuellen Einstellungen entsprechend - mit dem Gitter ausgerichtet. Auch die Kurve selbst wird bei der Erstellung am Gitter ausgerichtet.

Wenn Sie nur einige Punkte der Kurve am Raster ausrichten möchten, wählen Sie diese aus und wählen Kurve - > Ausgewählte Punkte am Raster ausrichten.

Konvertieren von und zu normalen Kurven

Bis jetzt besitzen Advanced Curves keine neuen Features - nur Ihr Editier- und Erstellungsmechanismus ist anders als bei herkömmlichen Kurven. Das bedeutet das eine existierende Kurve in eine Advanced Kurve umgewandelt werden kann und umgekehrt ohne irgendwelche Informationen zu verlieren. Wählen Sie einfach das zu konvertierende Objekt aus und wählen Sie "Objekt" -> "Kurventyp umwandeln (erweitert/normal)".

Sie können auch Tubes in AdvCurves konvertieren. Bitte beachten Sie, das Sie dabei die Dickeninformation der Tube (Röhre) verlieren.

Benutzen von Scripten mit AdvCurves

Für externe Scripte sieht eine AdvCurves wie eine normale Kurve aus, wenn es nicht über den Kurventyp Bescheid weiß - deshalb wird jedes dieser Scripte die Kurve automatisch in eine normale Kurve konvertieren - im Normalfall wird es jedenfalls arbeiten.

Warum wird ein Pfeil am Ende der Kurve angezeigt?

Wenn Sie einfache Objekte aus einer Kurve generieren (z.B. eine Röhre), müssen Sie nicht wissen, wo die Kurve anfängt oder aufhört. Aber wenn Sie eine Vervielfältigung (Array), eine Extrusion einer Kurve an einer anderen Kurve, Hautüberzug oder anders ausführen wollen, dann ist es sinnvoll zu wissen wo die Kurve anfängt, oder ob alle Kurven eines Skinnings (Überzug) übereinstimmend ausgerichtet sind.
